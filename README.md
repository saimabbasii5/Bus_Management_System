# Bus Management System

## Overview
The Bus Management System is a software application developed to manage bus operations efficiently. It helps in maintaining bus records, route information, schedules, and passenger details in a centralized system.

## Features
- Add, update, and delete bus,drivers records
- Manage bus routes and schedules
- Ticket Booking
- Store passenger information
- Search and view records easily
- User-friendly interface
- Efficient data management

## Technologies Used
- Programming Language: [HTML/CSS/PHP]
- Database: [MySQL] 
- IDE: [Visual Studio]
